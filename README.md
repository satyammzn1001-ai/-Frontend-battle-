# 🚀 Kepler AI

**Automate Everything. Babysit Nothing.**

Kepler AI is a modern AI workflow automation platform that helps teams automate repetitive tasks, monitor workflows, and improve productivity through an intuitive and responsive interface.

## ✨ Highlights

- 🤖 AI-powered workflow automation
- 📊 Interactive analytics dashboard
- ⚡ Modern, fast & responsive UI
- 🎨 Clean user experience with smooth animations
- 🌙 Built for scalability and performance

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- JavaScript
- Framer Motion
- Vercel
